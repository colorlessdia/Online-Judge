### 문제 설명

<p>Перед доктором Стрэнджем на столе стоит несколько граненых стаканов. Каждый из них при взгляде сверху выглядит как выпуклый многоугольник, а его стенки вертикальны и перпендикулярны столу. </p>

<p>Перед медитацией Стрэндж хочет распределить по стаканам некоторый объем воды, причем так, чтобы высота воды во всех стаканах была одинакова. Гарантируется, что стаканы достаточно высокие, чтобы так можно было сделать.</p>

<p>Помогите ему определить, какой уровень воды должен быть во всех стаканах.</p>
